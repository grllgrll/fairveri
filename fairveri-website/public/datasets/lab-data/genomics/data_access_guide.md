# Türk Popülasyonu Genom Projesi - Veri Erişim Kılavuzu

## 🔒 Kontrollü Erişim Politikası

Bu genomik veri seti, katılımcıların mahremiyetini korumak ve etik kullanımı sağlamak amacıyla **kontrollü erişim** politikası altında yönetilmektedir.

## 📋 Erişim Gereksinimleri

### 1. Araştırmacı Eligibilitesi
- Tanınmış bir araştırma kurumuna bağlılık
- Genomik araştırma deneyimi
- Etik onay belgesi
- Kurumsal destekleyici mektup

### 2. Proje Kriterleri
- Bilimsel değer taşıyan araştırma projesi
- Türk popülasyonu sağlığına katkı potansiyeli
- Etik kurallara uygun araştırma tasarımı
- Katılımcı haklarını gözeten yaklaşım

## 🔐 Veri Güvenliği Standartları

### Teknik Güvenlik
- **Şifreleme**: AES-256 bit encryption
- **Erişim kontrolü**: Multi-factor authentication
- **Ağ güvenliği**: VPN bağlantısı zorunlu

### Fiziksel Güvenlik
- Güvenli veri merkezleri
- Yedekleme sistemleri
- Felaket kurtarma planı
- ISO 27001 sertifikası

## 📊 Veri Formatları ve Standartları

### Genomik Veri Formatları
- **VCF**: Variant Call Format (v4.3)
- **CRAM**: Compressed Reference-oriented Alignment Map
- **GVCF**: Genomic Variant Call Format
- **HDF5**: Hierarchical Data Format

### Metadata Standartları
- **GA4GH**: Global Alliance standards
- **Phenopackets**: v2.0 phenotypic data
- **OMOP CDM**: Common Data Model
- **HL7 FHIR**: Healthcare interoperability

## 🌐 Erişim Seviyeleri

### Seviye 1: Özet İstatistikler
- **Erişim**: Açık erişim
- **İçerik**: Popülasyon frekansları, genel istatistikler
- **Format**: Web dashboard
- **Kayıt**: Basit kayıt yeterli

### Seviye 2: Gruplandırılmış Veriler
- **Erişim**: Kayıtlı araştırmacılar
- **İçerik**: Bölgesel/demografik gruplar
- **Format**: Özetlenmiş tablolar
- **Kayıt**: Araştırmacı kaydı gerekli

### Seviye 3: Bireysel Genom Verileri
- **Erişim**: DAC onayı gerekli
- **İçerik**: Tam genom dizileri
- **Format**: VCF, CRAM dosyaları
- **Kayıt**: Kapsamlı başvuru süreci
