## Deploy to Vercel (5 minutes):

1. **Install Vercel CLI:**
   ```bash
   npm install -g vercel
   ```

2. **Deploy:**
   ```bash
   vercel --prod
   ```

3. **Follow prompts:**
   - Link to your GitHub account
   - Choose project name
   - Deploy

You'll get a live URL like: `https://fairveri-website.vercel.app`
